# Creative Lineage

This document tracks how each experiment contributes to the evolving creative system behind juan.software.

## resonance
- Introduced particle systems as expressive matter
- Established controller-based architecture and animation loop
- Explored motion, inertia, and visual rhythm

## mirror
- Introduces camera as primary input
- Adds person segmentation and presence detection
- Transforms particles from abstract fields to body-driven matter
- Begins separation between input, processing, interpretation, and rendering

## Principle
Each new experiment should:
- reuse at least one meaningful system from a previous one
- improve or evolve it
- leave behind something reusable

This lineage is not about features.
It is about evolving a language of interaction.
